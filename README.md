# chip8-emulator

My version of Chip 8 Emulator with sound using SDL2.0

![](https://media.giphy.com/media/dZiI0IWx4cLGGDo0UD/giphy.gif)

## Install Instructions:
  1. Download and unpack the repo.
  2. Go to repo folder and type make.
  
## Instructions to use emulator
  1. Keep the games with .ch8 extension in roms folder.
  2. use up and down arrows to scroll and Enter Key to play the selected game.
  3. use left and right arrows to switch between menus.
